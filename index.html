SOBRE A COLEÇÃO

Os Micro-jogos CRIACOM são pequenas experiências interativas — cinco minutos ou menos cada — que demonstram princípios da Análise do Comportamento. Cada volume isola um conceito comportamental e o transforma em mecânica jogável.

Extinção, discriminação espacial, indução, insight, desconto temporal, esquemas de reforçamento... cada cartucho é uma microintuição sobre os processos que governam o que você faz — agora mesmo, lendo isto.

Essa interface entre game design e análise do comportamento é o que chamamos de Behavioral Game Design: usar a linguagem dos jogos para tornar visíveis as variáveis que regem o comportamento.
